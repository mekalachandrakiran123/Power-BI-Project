# Power-BI-Project
## Customer Churn Analysis using Power BI
### Project Overview

Customer churn is a major challenge for telecom companies, as acquiring new customers is significantly more expensive than retaining existing ones.
This project uses Power BI to analyze customer behavior, identify churn patterns, and provide actionable insights to reduce customer attrition.

The analysis transforms raw customer data into interactive dashboards that help understand why customers leave, who is most at risk, and what actions can improve retention.

### Objectives

Analyze customer churn patterns using historical data

Identify key drivers influencing customer churn

Visualize churn trends through interactive dashboards

Provide data-driven recommendations to improve customer retention

### Dataset Description

Total Records: 7,043 customers

Total Features: 21 columns

### Key Data Categories:

Customer Profile: Senior citizen, partner, dependents

Services: Phone service, internet type, streaming services

Account Details: Contract type, tenure, payment method

Financials: Monthly charges, total charges

Target Variable: Churn (Yes / No)

### Data Preparation & Cleaning

The following steps were performed to ensure data quality:

Fixed missing values (11 records in Total Charges)

Converted text-based numeric columns into numerical format

Removed unnecessary spaces and inconsistencies

Verified duplicate records (none found)

Ensured clean, structured data for accurate analysis

#### Key Drivers of Customer Churn

##### Overall churn rate: 26.54%

##### Month-to-month contracts are high risk

First-year customers need immediate retention strategies

##### Fiber optic service dissatisfaction

Electronic check payment method has high churn

Automatic payments improve customer loyalty

##### Long-term contracts significantly reduce churn

###  Applications & Recommendations

##### Encourage long-term contracts with incentives

Provide free or discounted tech support

Improve onboarding experience for new customers

##### Offer loyalty discounts to high-paying customers

##### Promote automatic payment methods

Use churn dashboards for proactive retention strategies

### Conclusion

Data-driven insights help reduce customer churn

Clean data leads to better decision-making

Retaining customers saves cost and increases revenue
